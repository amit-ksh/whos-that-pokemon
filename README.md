# WHO'S THAT POKEMON

Who's That Pokémon is a web app built with React, Node.JS and the Pokéapi to simulate the guessing game featured on the original Pokémon show during the commercial bumpers.

## SETUP

1. **BACKEND**

  ```js
    cd backend
    npm i
    npm start
  ```

1. **FRONTEND**

  ```js
    cd frontend
    npm i
    npm run dev
  ```
